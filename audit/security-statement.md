# Security Statement

This contract includes several early-stage protection mechanisms such as 
adjustable tax rates (with delay), adjustable limits, blacklist functionality, 
and a trading enable switch. These features are commonly used to ensure a fair 
launch and protect the market from automated attacks.

The owner commits to:
- Not abusing any privileged functions
- Using blacklist only against malicious bots
- Not modifying tax or limits in a harmful manner
- Not disabling trading after launch
- Maintaining transparency with the community

No minting, no hidden functions, and no ability to modify user balances exist.
